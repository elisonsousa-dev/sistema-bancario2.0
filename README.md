# Sistema Banário 2.0

Projeto desenvolvido em java com foco em Programação Orientada a Objetos.

## Funcionalidades
- Cadastro de usuários
- Criação de contas
- Despósitos
- Saques
- Transferências
- Consulta de saldo
- Consulta de dados

## Tecnologias utilizadas
- Java
- HashMap
- POO

## Observações
Projeto em evolução, com melhorias planejadas em validação e tratamentos de erros.
